我们通常推荐使用 <Link href="/new">vike.dev/new</Link> 来创建新的 Vike 应用，它会脚手架出使用 <UiFrameworkExtension name /> 的应用，以获得完整的 React/Vue/Solid 集成。

不过，你也可以创建一个不使用任何 Vike 扩展的应用，并自行实现自定义的 React/Vue/Solid 集成。

<CustomIntegrationWarning />

使用自定义 React/Vue 集成来脚手架一个 Vike 应用：

```shell
npm create vike-core@latest
```

可选项：
- `--skip-git`：不初始化新的 Git 仓库

随后你会在提示中选择以下之一：
- `react`:React + JavaScript
- `react-ts`:React + TypeScript
- `vue`:Vue + JavaScript
- `vue-ts`:Vue + TypeScript
