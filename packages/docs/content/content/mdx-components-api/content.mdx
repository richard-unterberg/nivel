The main package exports docs-friendly MDX components for content authors. Import them directly inside MDX files when needed.

```mdx
import { Alert, Link, Overview, RepoLink, Table } from '@unterberg/nivel'
```

## Component roles

- `Alert` renders callouts for notes and warnings.
- `Link` resolves docs-aware links and labels.
- `Overview` renders card grids for related docs pages.
- `RepoLink` creates consistent repository links.
- `Table` provides the styled table primitive.
